# CodeAlpha_BasicChatbot

## Description
A simple rule-based chatbot built using Python. The chatbot responds to predefined user messages.

## Features
- Greeting responses
- Basic conversation
- Exit command
- Console-based interaction

## Technologies Used
- Python
- Functions
- Loops
- Conditional Statements

## Sample Output

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
